Assixt

Usage: assixt [options] <command> [arguments]...
  assixt bootstrap config <option> [value]
  assixt [--dir=/usr/share/man] bootstrap man
  assixt bump [ patch | minor | major ]
  assixt [--no-files] [--no-meta] clean [path]
  assixt [--no-install] depend <module>...
  assixt dist [path]...
  assixt help
  assixt [--no-git] [--no-travis] new [name]
  assixt [--no-bump] push [path]...
  assixt test
  assixt touch bin <name>
  assixt touch class <name>
  assixt touch resource <name>
  assixt touch test <name>
  assixt touch unit <name>
  assixt [--pause-id=<id>] [--pause-password=<pass>] upload <dist>...

Options:
  -h --help             Show this help output.
  --config-file=<file>  Use a specified configuration file.
  --force               Try to continue, no matter what.
  --no-user-config      Don't use the user-specific configuration.
  --verbose             Add additional output.
